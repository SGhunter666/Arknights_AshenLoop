# 音效方向

这套音效先用程序生成，目标不是“真实拟音”，而是先把游戏的交互节奏和机制辨识度立起来。

## 整体方向

- 主菜单 / 页面按钮：偏石板与玻璃感，短、干净、略带厚度
- 战斗手牌：偏清脆纸牌与术式启动感
- Resonance：偏冷色晶体与相位回响
- Support：偏罗德岛指挥终端与队伍协同
- W：偏危险炸药与延时引爆警报
- 胜利 / 失败：简短结算旋律，不拖太长

## 混音分层

- `UI`
  - 页面打开、按钮点击、商店/休整/奖励进入
- `Combat`
  - 抽牌、选牌、出牌、命中、结束回合
- `Special`
  - 支援牌、共振叠层、共振引爆、胜利/失败
- `Warning`
  - W 炸药警报、爆炸

## 分层优先级

- `Warning` 会短暂压低 `UI / Combat / Special`
- `Special` 会短暂压低 `UI / Combat`
- `Combat` 和 `UI` 维持常态层

这样战斗里的重点音效会更突出，不会被普通按钮音和常规命中音盖住。

## cue 列表

- `ui_click.wav`
  - 主菜单、通用按钮、功能确认
  - 感受：石板轻敲
- `ui_open.wav`
  - 通用页面打开
  - 感受：轻薄玻璃展开
- `card_draw.wav`
  - 抽牌入手
  - 感受：纸牌掠过 + 轻闪
- `card_select.wav`
  - 选中手牌 / 进入瞄准
  - 感受：清脆术式定位
- `card_play.wav`
  - 普通出牌
  - 感受：甩牌出手
- `support_play.wav`
  - 支援牌
  - 感受：短促指挥回响，比普通出牌更亮
- `attack_hit.wav`
  - 命中
  - 感受：短促冲击
- `resonance_apply.wav`
  - 叠加共振
  - 感受：蓝白色晶体叠层
- `resonance_burst.wav`
  - 共振引爆 / 消耗
  - 感受：相位破裂
- `explosion_warn.wav`
  - W 炸药预警
  - 感受：急促双段提示
- `explosion_boom.wav`
  - 高爆 / 大伤害爆裂
  - 感受：闷爆
- `end_turn.wav`
  - 结束回合
  - 感受：印章落下
- `shop_open.wav`
  - 商店页
  - 感受：交易提示 + 轻铃
- `rest_open.wav`
  - 休整页
  - 感受：温暖、放松
- `reward_open.wav`
  - 奖励页
  - 感受：揭示感
- `victory.wav`
  - 胜利页
  - 感受：上扬三音
- `defeat.wav`
  - 失败页
  - 感受：下坠、收束

## 后续替换规则

如果以后你换成更精致的真实音效，优先保持：

- 文件名不变
- cue 语义不变
- 触发点不变

这样代码不需要改，只替换 `assets/sfx/` 里的资源就行。
